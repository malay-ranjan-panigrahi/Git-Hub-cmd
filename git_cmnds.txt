Git basic Commands:
-------------------------------------------------------------------------
git add <filename> #adding a file on stage area.
git rm --cached <filename> or git restore --staged <filename># unstage a file or make a staged file to untracked.
git commit -m "message " #make a file from staged to tracked.
git restore <file> # restore a deleted tracked file on local repo.

git remote add origin <repo link> # configure a exixting github repo on a local git repo.
or
git remote set-url roigin https://PAT@github.com/malay-ranjan-panigrahi/Git-Hub-cmd.git
git remote -v # to check targated remote repo.
git push origin main #push the code from local repo to the github repo.
git clone <remote repo link> # copyinh a remote repo to local repo.
git pull origin main #pulling changes from remote(github repo) to our local repo.

Branches:
-----------------------------------------------
git branch #it shows how many branches you have and in which branch you are in.
git checkout -b <branch name> #creating a new branche and switch to this new branch.
git switch <branch name> # switch from one branch to another branch
